## [0.1.0] - 2024-01-XX

### Added
- Initial project setup with Create React App
- React Router for navigation
- Shopping cart functionality with CartContext
- Product listing page with filtering
- Category browsing
- Hero slider component
- Material showcase gallery
- Admin dashboard
- Authentication system with JWT
- Billing/invoice system
- WhatsApp integration widget
- Price calculator
- Shipping estimator
- Inquiry form
- SEO optimization with React Helmet
- Responsive design for mobile, tablet, and desktop

### Features
- Product browsing by categories (Bathroom, Kitchen, etc.)
- Product filtering by type, color, and price
- Shopping cart with square footage calculation
- Real-time price calculation
- Tax and discount calculation
- Customer information collection
- Bill generation
- Admin authentication
- Product management

### Technical
- React 19.2.0
- React Router DOM 6.30.1
- Axios for API calls
- React Helmet Async for SEO
- LocalStorage for cart persistence
- JWT token-based authentication

## How to Update This Changelog

When making changes:

1. Add entries under `[Unreleased]` for new changes
2. When releasing a new version:
   - Change `[Unreleased]` to the new version number and date
   - Create a new `[Unreleased]` section
   - Update the version history

### Change Types

- **Added** - New features
- **Changed** - Changes in existing functionality
- **Deprecated** - Soon-to-be removed features
- **Removed** - Removed features
- **Fixed** - Bug fixes
- **Security** - Security fixes
